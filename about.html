About Me

My name is Justin Proffitt. This is my blog - where I write about what's on my mind and what I'm up to. It is generated using static, a static site generator written in Clojure.

I like to build things when I have the time to do so, you can checkout some of the things I've worked on the projects page and my github profile.

I'm interested in all things technical, but lately I have been focused on software security. Professionally I have been building things with Java, C, C++, Python, Javascript, and Groovy for 3+ years. In my free time I like to use Clojure and ClojureScript where appropriate, falling back to C or Python when necessary.

My non-technical interests are reading, music, movies, board games, playing soccer, hiking, and trivia.

If you're curious, the background color of this website is bone, and the font is something called silian rail.
